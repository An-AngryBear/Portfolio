About Jason

I'm a full-stack apprentice software developer with strong communication and problem-solving skills. I find joy in bashing my head against code until the problem is solved and learning more efficient ways to accomplish tasks.

My software development story began in late 2016 as I was exploring different careers paths. As a guy who has always had an interest in tech, naturally I came across software development. One thing led to another and I began to teach myself code by taking classes online. I immediately fell in love with the logic of JavaScript. A friend of mine recommended that I go to a software development bootcamp and hone my skills. I then began my path as a programmer by attending the Nashville Software School.

Projects

Patch Notes

A web application that utilizes the Steam API, so PC gamers can access their accounts, display their owned games, and find the recent updates for their favorite games. It was built not only with AngularJS, but Firebase, Sass, Grunt and Node.js proxy server.

Down2Game

A web application with the purpose of matching online gamers together based on personality traits, gaming habits, etc. It features a player matching system, live chat, a customizable profile, and access to the International Games Database. It was built using Node.js, PostGreSQL, Sequelize, Jquery, Socket.io, AWS S3, Grunt, and Pug.
